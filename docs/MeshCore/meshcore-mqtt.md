# MeshCore MQTT Server

## LetsMesh Network Analyzer
[LetsMesh](https://analyzer.letsmesh.net) provides real-time MeshCore analytics, node tracking, packet analysis, and coverage maps by collecting data from **Observer** nodes across the network.This mqtt service also talks to meshmapper for wardriving. 

### Becoming an Observer
To contribute to LetsMesh, your node needs to be set up as an Observer — a MeshCore repeater, room server, or companion device that reports heard packets to the service via MQTT.

1. Visit the [LetsMesh Observer Onboarding page](https://analyzer.letsmesh.net/observer/onboard)
2. Select your node type (repeater, room server, or companion)
3. Follow the on-screen steps to configure your Wi-Fi and MQTT uplink settings
4. Reboot — your node will begin reporting to LetsMesh within a few minutes
